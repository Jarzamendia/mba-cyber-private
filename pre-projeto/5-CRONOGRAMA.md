# Cronograma - Roteiro

## Objetivo desta seção
O cronograma indica a sequência de atividades e o tempo estimado para cada etapa do projeto. Segundo o template, deve respeitar o prazo máximo de entrega do artigo no AVA.

## Formato
Tabela com etapas nas linhas e semanas nas colunas. Use "X" para marcar as semanas de cada atividade.

## Notas
- Este cronograma segue exatamente o modelo do template institucional.
- As semanas devem ser ajustadas conforme as datas reais do AVA.
- A Etapa 2 é a mais longa pois inclui a triagem dos estudos, leitura integral dos trabalhos selecionados e a síntese comparativa dos achados.
- **Importante**: Verificar no AVA as datas exatas de cada entrega para ajustar as semanas.

## Dicas
- O cronograma do pré-projeto é uma **proposta** — pode ser ajustado depois.
- Mantenha-o simples e realista.
- Na versão final (Word/PDF), use uma tabela formatada conforme o template.
