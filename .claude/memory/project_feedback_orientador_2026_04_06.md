**Novo título definido pelo orientador:** "Shift-Left Security: Detecção de Vulnerabilidades em Docker Imagens"

**Requisitos obrigatórios por seção:**

### Introdução (6 pontos)
1. Visão geral sobre o tema
2. Relevância no contexto acadêmico, social ou tecnológico
3. Dados, citações ou referências para fortalecer importância
4. Lacuna ou desafio existente no tema
5. Por que essa lacuna precisa ser abordada
6. Problema específico que o trabalho pretende investigar

### Justificativa (5 pontos)
1. Por que o tema é importante (acadêmico, científico ou social)
2. Escolha do tema com base em relevância ou lacunas na literatura
3. Questões não suficientemente exploradas
4. Impacto que a revisão bibliográfica pode gerar
5. Demonstrar que revisão bibliográfica é método adequado e há material suficiente

### Metodologia (9 pontos)
1. Especificar que é revisão bibliográfica
2. Bases de dados consultadas + motivos
3. Intervalo de tempo + justificativa
4. Parâmetros (idioma, tipo de publicação, relevância temática)
5. Triagem dos estudos
6. Etapas (títulos, resumos, textos completos)
7. Métodos/critérios de análise e interpretação

**Preocupação do orientador:** Se haverá bibliografia suficiente para o tema específico.

**Why:** O orientador está validando se o João consegue adaptar o trabalho para o formato obrigatório de revisão bibliográfica.
**How to apply:** Cada seção precisa ser revisada para atender explicitamente cada ponto listado. A metodologia precisa reescrita completa. A justificativa precisa de um parágrafo novo justificando o método de revisão bibliográfica.
